# golang-im-system

A simple golang instant message system.

## Build

### Windows

```bash
build.bat
```

## Usage

### Windows

Firstly, start the server:

```bash
server
```

Link to server in windows shell:

```bash
nc 127.0.0.1 8888
```

Repeat it. You can mock multi user online.

To client, you can get help from:

```bash
client -h
```
